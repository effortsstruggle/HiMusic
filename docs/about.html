使用说明

使用前请确保安装了 Python 3 以上版本。

安装相关库

在安装根目录下进入命令行，输入以下命令，回车运行：

  1. pip install -r requirements.txt

或者运行根目录下的 requirements.bat 批处理文件安装。

运行

双击安装文件夹下的 main.py 即可运行。
